# Objetivo de negocio

Este dashboard se está desarrollando para presentar una propuesta seria de Dirección de Data dentro de la Secretaría del Deporte de la Ciudad de México.

El objetivo no es solo mostrar visualizaciones, sino demostrar una capacidad real para:
- integrar datos públicos deportivos,
- generar KPIs accionables,
- visualizar brechas territoriales por alcaldía,
- relacionar actividad física, infraestructura y salud,
- construir una base escalable para una dirección institucional de análisis de datos.

La prioridad inmediata es tener un MVP sólido, visualmente profesional y técnicamente defendible.
